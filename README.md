Sitio Integrador II - Full Stack App 
Descripción
Este proyecto es el resultado final de mi Diplomatura en Programación Web Full Stack en la UTN.BA. Consiste en una plataforma web dinámica que integra un frontend moderno con una arquitectura de backend robusta y gestión de base de datos relacional.   

Tecnologías Utilizadas 
Frontend: React.js , Next.js , HTML5, CSS3 y JavaScript (ES6+).   
Backend: Node.js y Express.js.   
Base de Datos: SQL (MySQL/MariaDB) con modelado de datos relacional.   
Herramientas: Git para control de versiones y DBeaver para la gestión de la base de datos.   

Características Principales
Arquitectura de Componentes: Desarrollo de interfaces modulares y reutilizables en React.   
API REST: Implementación de rutas y controladores en Node.js para el manejo de peticiones.   
Persistencia de Datos: Operaciones CRUD completas sobre una base de datos SQL.   

Lógica de Negocio: Resolución de problemas lógicos aplicados al flujo del usuario.
